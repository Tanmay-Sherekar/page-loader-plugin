# 🚀 Advanced Page Loader Plugin (WordPress)

A premium WordPress plugin that adds a **curved shutter opening loader** with **animated developer icons** and **loading text** to all pages.

---

## 🎬 Features

- 🎥 Curved shutter opening animation (cinematic effect)
- ⚫ Full black background loader
- 💻 Animated developer icons (floating effect)
- ⏳ Loading text with pulse animation
- ⚡ Smooth transition on page load
- 🧩 Lightweight & easy to integrate

---

## 📸 Preview

> Loader appears on page load with animated icons and disappears with a curved shutter opening effect.
